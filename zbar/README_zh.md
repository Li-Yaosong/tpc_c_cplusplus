#  zbar 三方库说明

## 功能简介

zbar是一个条形码和二维码解析的库。

## 使用约束

- Rom版本：OpenHarmony3.2 beta2
- IDE版本：DevEco Studio 3.1 Beta1
- API版本：9
- 三方库版本：0.10
- 当前适配的功能：条形码和二维码解析
- License：[LGPL](https://github.com/herbyme/zbar/blob/master/COPYING)
